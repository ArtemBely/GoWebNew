Transportability

The computing centres themselves are built
in a way that allows GoWeb to considerably
decrease costs of computing power for AI.

The operating system enables equipment network
booting (PXE) without using a hard disk drive
(HDD/ SSD). This booting technique enables
to quickly update key system components, drivers
and software on all computing systems.
The advantage is that it simplifies equipment
administration and lowers the risk of an outage.

The company not only develops and manufactures
its own equipment but also develops software
for maintaining the computation process
and monitoring equipment efficiency rate.

Why the modular centre

  • Setting up a data centre with more than 300 machnes requires both high professionalism and safety measures.

  • Security. The modular centres are placed in security guarded areas.

  • Protection. Each modular centre is equipped with an effective fire protection system.

  • Mobility. The rigid skeleton inside that allows the container to be transported anywhere

  • Capacity. It is easy to increase the capacity of the computing centre at any point in time.

  • Modules: It allows to create of an accumulated computing power solution

  • Cooling: high tech cooling system is working at each module. Details can be given upon request

  • Pricing: flexible and welcoming pricing strategy will cover the transitional cost for the companies and let the business get to know each other.
